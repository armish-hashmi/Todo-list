# To-Do List App

A simple React to-do list — add tasks, mark them done, delete them.

## Setup

```bash
npm install
npm start
```

Then open http://localhost:3000

## What's inside

- `src/App.js` — all the app logic (add/toggle/delete tasks)
- `src/App.css` — styling
- Tasks are stored in memory only — refreshing the page clears the list
